# Asplenie – Rechtsdokumente

Dieses Repo hostet ausschließlich die öffentlich zugänglichen Rechtsdokumente
der iOS-App **Asplenie**:

- [`privacy.html`](./privacy.html) – Datenschutzerklärung
- (folgt) Impressum, ggf. Nutzungsbedingungen

Der App-Quellcode liegt in einem separaten, privaten Repository.

## Veröffentlichung

Diese Dateien werden via **GitHub Pages** unter
<https://awittemann.github.io/asplenie-legal/> bereitgestellt und in der App
sowie in App Store Connect als Privacy-Policy-URL referenziert.
